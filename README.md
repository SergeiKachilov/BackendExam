# ОНЛАЙН МАГАЗИН ЭЛЕКТРОНИКИ (3 вариант)
## Описание
Проект является backend-приложением для онлайн-магазина электроники  
Проект позволяет:

- Покупателям – просматривать товары, создавать заказы, оставлять отзывы
- Менеджерам – управлять товарами и заказами пользователей
- Админам – предоставляется полный доступ.

## Инструкция по запуску
1. Создание venv (если нет папки .venv)  
`py -m venv .venv`

2. Активация venv (находясь в командной строке в директории проекта)  
`.venv\Scripts\activate`

3. Установка зависимостей  
`pip install -r requirements.txt`

4. Создание и посев базы данных  
`python seed.py`

5. Запуск проекта:
    1. В режиме dev  
    `python -m fastapi dev main.py`
    2. В режиме prod  
    `python -m fastapi run main.py`

## Описание API
- **Методология:** REST  
- **Формат передачи данных:** JSON  
- **Документация:**  
    `http://127.0.0.1:8000/docs`  
    (Проект должен быть запущен)  

Аутентификация происходит с помощью jwt токена с возможностью обновления через refresh токен

## Валидация
- **Email:** проверяется регулярным выражением
- **Пароль:** Длина минимум 8 символов
- **Логин:** Длина минимум 5 символов
- **Уникальность:** `email`, `логин` должны быть уникальными у каждого пользователя, `название` уникально для каждого товара
- **Ограничение по товарам в корзине:** количество добавляемых товаров в корзину не может быть больше доступного количества на складе, товар не будет добавлен, если он отсутствует на складе
- **Ошибки валидации** возвращают ответ с различными HTTP-статусами (403, 404, 409, 422)

## Роли
- **Пользователь:** просмотр товаров, изменение своей корзины, создание заказа, изменение своих данных (кроме роли); создание, изменение и удаление своих отзывов
- **Менеджер:** Управление и изменение товаров (за исключением цены), изменение информации о заказах пользователей
- **Администратор:** Имеет полный доступ, за исключением доступа к данным других администраторов, имеет возможность удалять чужие комментарии (с указанием причины)
